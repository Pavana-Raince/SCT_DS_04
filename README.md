# SCT_DS_04
## Task 04 – UK Road Accident Analysis
## Objective
Analyze UK road accident data (2005–2015) to identify patterns related to road conditions, weather, and time of day. 
Visualize accident hotspots and contributing factors.

## Tools Used
Power BI Desktop

DAX (Data Analysis Expressions)

Dataset: UK Road Safety Data (Accidents0515.csv),(https://www.kaggle.com/datasets/tsiaras/uk-road-safety-accidents-and-vehicles)

## Description
Performed Exploratory Data Analysis (EDA) to transform raw geographic and temporal data into an interactive dashboard. 
Addressed technical challenges including DAX circular dependencies and data type conversions to enable accurate chronological sorting of accident trends.

## Files Included
road_accident_analysis.pbix – Power BI project file

Accidents0515.csv – Source dataset

output_chart.png – Final dashboard visualization

## Result
Identified geographic hotspots and revealed that most incidents occur during Fine weather on Dry roads.
Temporal analysis highlights a significant frequency spike during Afternoon and Evening periods, correlating with peak commuting hours.
